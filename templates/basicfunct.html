Python Lambda Functions

What Is a Lambda?

A lambda is an anonymous function â€” a quick, one-line function without a name.

Defining a Lambda

lambda arguments: expression

Example

square = lambda x: x ** 2
print(square(4))  # Output: 16

Using Lambdas in Self-Defined Functions

You can pass a lambda function into your own function.

def apply_operation(x, func):
    return func(x)

result = apply_operation(10, lambda n: n + 5)
print(result)  # Output: 15

The map() Function

map() applies a function to each element in an iterable.

nums = [1, 2, 3, 4]
squared = list(map(lambda x: x ** 2, nums))
print(squared)  # Output: [1, 4, 9, 16]

filter() selects elements that return True when passed to the function.

nums = [1, 2, 3, 4, 5]
evens = list(filter(lambda x: x % 2 == 0, nums))
print(evens)  # Output: [2, 4]
