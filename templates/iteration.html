Looping and Flow Control in Python

The pass Instruction

pass is a placeholder. It does nothing, but it helps you write code that’s syntactically correct when a block is required.


if True:
    pass  # Will not raise an error, even though nothing happens

Loops: while, for, range()

while Loop

Repeats as long as the condition is True.


count = 0
while count < 3:
    print(count)
    count += 1

for Loop with range()

Executes a block of code a fixed number of times.


for i in range(3):
    print(i)

Iterating Through Sequences

You can loop over strings, lists, or other sequences using for ... in.


fruits = ["apple", "banana", "cherry"]
for fruit in fruits:
    print(fruit)

Loop + else: while-else and for-else

The else block runs if the loop wasn’t stopped by a break.

while-else Example


i = 0
while i < 3:
    print(i)
    i += 1
else:
    print("Done!")

for-else Example


for n in range(3):
    print(n)
else:
    print("Loop finished.")

Nesting Loops and Conditionals

Loops and if statements can be placed inside each other.


for i in range(1, 4):
    if i % 2 == 0:
        print(f"{i} is even")
    else:
        print(f"{i} is odd")

Controlling Loops with break and continue

break – exit the loop early


for i in range(5):
    if i == 3:
        break
    print(i)  # Output: 0, 1, 2

continue – skip to next iteration


for i in range(5):
    if i == 3:
        continue
    print(i)  # Output: 0, 1, 2, 4

Tips

  • Use range(start, stop) to loop over numbers
  • for is best when you know how many times to loop
  • while is best when the end condition depends on user input or logic
  • Use break to exit early, continue to skip a step
  • else on loops is rare, but useful when checking if something was found
