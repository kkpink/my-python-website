Python Input & Output Functions

Output with print()

The print() function displays messages or values to the screen.

print("Hello, world!")
print("Name:", "Alice", sep=" - ")

Customizing print()

  • sep= defines the separator between values
  • end= changes what comes at the end (default is newline)
=print("Python", "Rocks", sep="*", end="!!!\n")

Input with input()

The input() function asks the user for input and returns it as a string.

name = input("What is your name? ")
print("Hi,", name)

If you expect numbers, convert the input:

age = int(input("Enter your age: "))
print("You will be", age + 1, "next year.")

Tips

  • input() always returns a string
  • Use int(), float(), or str()to convert input as needed
  • Use print()to show output clearly and with context
