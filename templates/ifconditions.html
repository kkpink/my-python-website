Making Decisions in Python: if Statements

Python lets your code make decisions using conditional statements. These allow your program to react differently based on conditions or inputs.

Basic if Statement

Executes code only if the condition is True.


age = 20
if age >= 18:
    print("You are an adult.")

if-else Statement

Provides an alternative when the condition is False.


if age >= 18:
    print("Adult")
else:
    print("Minor")

if-elif Statement

Checks multiple conditions one by one. Only the first True block runs.


grade = 85

if grade >= 90:
    print("A")
elif grade >= 80:
    print("B")
elif grade >= 70:
    print("C")
else:
    print("F")

Multiple Conditional Statements

You can write multiple independent if statements — each will be checked and possibly executed.


num = 6

if num > 0:
    print("Positive")
if num % 2 == 0:
    print("Even")

Nesting Conditional Statements

Conditionals can be placed inside one another to check multiple levels.


score = 90

if score >= 60:
    print("You passed.")
    if score >= 90:
        print("Excellent!")
else:
    print("You failed.")

Best Practices

  • Use clear, readable conditions
  • Use elif for exclusive choices
  • Don’t over-nest; use and/or when possible
  • Indent consistently (PEP 8 recommends 4 spaces)
