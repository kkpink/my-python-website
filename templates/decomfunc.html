Defining and Using Functions in Python

User-Defined Functions

You can create your own functions using the def keyword. Functions let you reuse code and organize logic clearly.


def greet(name):
    print(f"Hello, {name}!")

greet("Alice")  # Calling (invoking) the function

The return Keyword

return sends a result back from the function to the caller. The function ends as soon as return runs.


def add(a, b):
    return a + b

result = add(5, 3)
print(result)  # Output: 8

The None Keyword

If a function does not use return, it returns None by default — which means “nothing.”


def say_hi():
    print("Hi!")

output = say_hi()
print(output)  # Output: None

Generators

Generators are a special type of function that uses yield instead of return. They return values one at a time, saving memory.


def count_up_to(n):
    count = 1
    while count <= n:
        yield count
        count += 1

for num in count_up_to(3):
    print(num)
# Output: 1 2 3

Recursion

A recursive function calls itself to solve a smaller part of the problem. It must have a base case to avoid infinite loops.


def factorial(n):
    if n == 1:
        return 1
    return n * factorial(n - 1)

print(factorial(5))  # Output: 120

Key Tips

  • Use functions to organize and reuse code.
  • return ends a function and sends back a value.
  • None means “no value” or “nothing returned.”
  • Generators are great for large loops or streams of data.
  • Recursion is powerful but must be written carefully to avoid infinite loops.
