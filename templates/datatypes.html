Python Data Types

Python has several built-in data types used to store different kinds of values. Here are the most common ones:

Integer (int)

Represents whole numbers, positive or negative.

age = 25
score = -10

Floating-Point (float)

Represents numbers with decimal points.

price = 19.99
temperature = -5.6

Boolean (bool)

Represents truth values: either True or False

is_active = True
has_permission = False

String (str)

Used to store text enclosed in quotes.

name = "Alice"
message = 'Hello, world!'

Scientific Notation (as float)

Used for very large or small numbers.

light_speed = 3e8  # 3 × 10⁸
mass = 9.11e-31

Binary, Octal, Hexadecimal

  • Binary: b1010→ 10
  • Octal: 0o17→ 15
  • Hexadecimal: 0x1F → 31
binary_number = 0b1010
octal_number = 0o17
hex_number = 0x1F
