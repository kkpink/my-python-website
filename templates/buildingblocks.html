Core Concepts: Data Types, Variables & Naming

Here we will disscus vital topics which ease your Python journey with Profesor Pytho!

Data Types

  • Boolean: Represents truth values, either True or False
  • Integers: Whole numbers like 11, 1 or 3.
  • Floating-point Numbers: Numbers with decimals, like 3.14 or 5.36.
  • Scientific Notation: Is used to represent very large or small numbers, like 1.5e6 which is (1.5 × 10⁶).
  • Strings: Sequences of text or charcters inside quotes, like "Hello World".

Numeral Systems and Bases

  • Binary (Base 2): Uses only 0 and 1, like 01010 which represents 10.Binary numbers in python have a prefix of Ob
  • Octal (Base 8): Uses digits 0–7, like 0o17.Octal numbers in Python have a prefix of Oo
  • Decimal (Base 10): Our usual number system which digits 0-9 like 99 or 58.
  • Hexadecimal (Base 16): Uses digits 0–9 and letters A–F, like 0x1F

Variables

  • Used to store values (like boxes with labels) as it acts like a container.
  • Example: name = "Alice"or core = 100.

Naming Conventions

  • Use lowercase letters and underscores: user_name, total_score.
  • Variable names should be descriptive: avoid x, y unless in math.
  • Names can’t start with numbers or use special symbols (except underscores).

PEP-8 Tips for Beginners

  • PEP-8 is the style guide for Python code.
  • Use 4 spaces for indentation (don’t mix tabs and spaces).
  • Keep lines under 79 characters for readability.
  • Leave a blank line between functions and classes.
  • Use meaningful names and consistent formatting.
